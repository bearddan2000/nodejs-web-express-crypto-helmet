# nodejs-web-express-crypto-helmet

## Description
An overall package for secure express
app. Removes self identifing headers.

## Tech stack
- nodejs
- npm
  - express
  - helmet

## Docker stack
- node:latest

## To run
`sudo ./install.sh -u`
- http://localhost

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
